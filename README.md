# Template
pugやscssでマークアップの練習をする用

## Getting Started

```
npm i
npm run dev
```

### templateからcloneする流れ
1. templateからclone
2. 新しくpushする予定のリポジトリを作成
3. git pushの指定先を、新しくpushする予定のリポジトリに変更する
4. git remote -vでpush先が変更できていることを確認。
5. remoteにcloneしたコードをコミット
6. 開発開始！
